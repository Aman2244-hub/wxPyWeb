5.5 Possible extensions

Of course, this program is far from being a decent editor. But adding other features should not be more difficult than what has already been done. You might take inspiration in the demoes that are bundled with wxPython:
  • Drag and Drog.
  • MDI
  • Tabview/multifiles
  • Find/Replace dialog
  • Print dialog
  • Macro-commands in python ( using the eval function)
  • etc ...
