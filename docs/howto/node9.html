4 A first application - Hello, world

As is traditional, we are first going to write a Small "Hello, world" application. Here is the code: 
    from wxPython.wx import wxPySimpleApp, wxFrame
    app = wxPySimpleApp()
    frame = wxFrame(None, -1, "Hello World")
    frame.Show(1)
    app.MainLoop()

Figure: Hello, World application
