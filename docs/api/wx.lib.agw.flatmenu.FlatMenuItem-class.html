Default class constructor.

Parameters:
Parameters:
parent - menu that the menu item belongs to;

helpString - optional help string that will be shown on the status bar;

kind - may be wx.ITEM_SEPARATOR, wx.ITEM_NORMAL, wx.ITEM_CHECK or wx.ITEM_RADIO;

normalBmp - normal bitmap to draw to the side of the text, this bitmap is used when the menu is enabled;

disabledBmp - 'greyed' bitmap to draw to the side of the text, this bitmap is used when the menu is disabled, if none supplied normal is used;

hotBmp - hot bitmap to draw to the side of the text, this bitmap is used when the menu is hovered, if non supplied, normal is used.
