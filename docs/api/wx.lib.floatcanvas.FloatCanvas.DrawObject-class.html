Class DrawObject

Known Subclasses:
Arrow, ArrowLine, Bitmap, Group, Line, Point, PointSet, Polygon, RectEllipse, ScaledBitmap, ScaledBitmap2, ScaledText, ScaledTextBox, SquarePoint, Text
!

This is the base class for all the objects that can be drawn.

One must subclass from this (and an assortment of Mixins) to create a new DrawObject.

ote This class contain a series of static dictionaries:

  • BrushList
  • PenList
  • FillStyleList
  • LineStyleList
