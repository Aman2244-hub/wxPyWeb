Package wx.tools

Some useful tools and utilities for wxPython.

Submodules

  • helpviewer: helpviewer.py -- Displays HTML Help in a wxHtmlHelpController window.
  • img2img
  • img2png: img2png.py -- convert several image formats to PNG format.
