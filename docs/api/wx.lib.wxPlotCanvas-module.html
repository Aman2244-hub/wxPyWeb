Module wx.lib.wxPlotCanvas

This is a port of Konrad Hinsen's tkPlotCanvas.py plotting module. After thinking long and hard I came up with the name "wxPlotCanvas.py".

This file contains two parts; first the re-usable library stuff, then, after a "if __name__=='__main__'" test, a simple frame and a few default plots for testing.

Harm van der Heijden, feb 1999
