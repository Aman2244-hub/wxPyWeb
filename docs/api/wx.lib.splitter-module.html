Module wx.lib.splitter

This module provides the MultiSplitterWindow class, which is very similar to the standard wx.SplitterWindow except it can be split more than once.

Classes
MultiSplitterEvent This event class is almost the same as wx.SplitterEvent except it adds an accessor for the sash index that is being changed.
MultiSplitterWindow This class is very similar to wx.SplitterWindow except that it allows for more than two windows and more than one sash.
