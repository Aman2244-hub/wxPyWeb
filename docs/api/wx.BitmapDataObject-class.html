Known Subclasses:
PyBitmapDataObject

wx.BitmapDataObject is a specialization of wxDataObject for bitmap data. It can be used without change to paste data into the wx.Clipboard or a wx.DropSource.

See Also:

wx.PyBitmapDataObject if you wish to override GetBitmap to increase efficiency.

GetBitmap(self)

Returns the bitmap associated with the data object. You may wish to override this method (by deriving from wx.PyBitmapDataObject) when offering data on-demand, but this is not required by wxWidgets' internals. Use this method to get data in bitmap form from the wx.Clipboard.

Returns:
Bitmap

SetBitmap(self, bitmap)

Sets the bitmap associated with the data object. This method is called when the data object receives data. Usually there will be no reason to override this function.
