Type GraphicsPath

    object --+    
             |    
GraphicsObject --+
                 |
                GraphicsPath

A GraphicsPath is a representaion of a geometric path, essentially a collection of lines and curves. Paths can be used to define areas to be stroked and filled on a GraphicsContext.
