                            FlatNotebook

Display one or more windows in a notebook.

Events:
  • EVT_FLATNOTEBOOK_PAGE_CHANGING: sent when the active page in the notebook is changing
  • EVT_FLATNOTEBOOK_PAGE_CHANGED: sent when the active page in the notebook has changed
  • EVT_FLATNOTEBOOK_PAGE_CLOSING: sent when a page in the notebook is closing
  • EVT_FLATNOTEBOOK_PAGE_CLOSED: sent when a page in the notebook has been closed
  • EVT_FLATNOTEBOOK_PAGE_CONTEXT_MENU: sent when the user clicks a tab in the notebook with the right mouse button
